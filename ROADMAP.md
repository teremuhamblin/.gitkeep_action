# 🗺️ ROADMAP — .gitkeep_action

Projet minimaliste basé sur `.gitkeep`, conçu pour structurer, organiser et automatiser des dossiers vides dans un workflow Git/GitHub.

---

## ✅ v1.0.0 — Initialisation du projet
- [x] Création de la structure de base
- [x] Ajout des dossiers principaux (src, logs, data, cache, tmp, docs, tests)
- [x] Ajout des `.gitkeep` pour versionner les dossiers vides
- [x] Création du README initial
- [x] Mise en place du `.gitignore`

---

## 🚀 v2.0.0 — Gestion avancée des dossiers
- [x] Ajout de descriptions enrichies dans chaque `.gitkeep`
